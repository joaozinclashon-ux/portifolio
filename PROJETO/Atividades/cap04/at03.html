Seção 2 — Ranking de Tecnologias (Top 5)

  1. HTML5

    A base de tudo. Sem ele, não existe estrutura na web.

  2. CSS3

    Essencial para o design e a responsividade das páginas.

  3. JavaScript

    A linguagem que traz interatividade e lógica ao navegador.

  4. React

    Uma das bibliotecas mais utilizadas no mercado atual.

  5. Node.js

    Permite rodar JavaScript no lado do servidor (Backend).

Seção 3 — Glossário de Termos Técnicos

HTML
HyperText Markup Language.
Linguagem de marcação padrão para criar páginas web.
JS
JavaScript
Linguagem de programação de alto nível executada principalmente em navegadores.
CSS
Cascading Style Sheets, usada para estilizar elementos HTML.
DOM
Document Object Model, a interface de programação para documentos HTML.
HTTP
HTTPS
Protocolos de transferência de dados na web.
URL
Uniform Resource Locator, o endereço de um recurso na rede.
